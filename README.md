# ntd-to-socrata-bts

Uploads GTFS data from the NTD (National Transit Database) to the https://data.bts.gov catalog

# Running the script

Set the two environment variables `SOCRATA_BTS_USERNAME` and `SOCRATA_BTS_PASSWORD`

```
python3 publish_to_catalog.py
```
